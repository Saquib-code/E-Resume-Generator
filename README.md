# E-Resume-Generator
E-Resume Generator | Spring Boot, Spring AI, React, Tailwind CSS, JavaScript, OpenAI API, OAuth2,AWS EC2
• Developed an AI-based resume builder where users input prompts to extract details like name, skills, and
experience using Spring AI with the ChatGPT API.
• Built an editable React + Tailwind interface with add/remove options for skills, languages, and experience before
generating PDFs.
• Secured the app with Spring Security and OAuth2 for Google and GitHub logins.
• Designed REST APIs in Spring Boot for prompt handling and structured data exchange between backend and
frontend.
• Deployed the application on AWS EC2, ensuring scalability and smooth performance.
